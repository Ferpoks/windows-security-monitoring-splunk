# Windows Security Monitoring using Splunk

This project demonstrates a Mini SIEM environment built using Splunk Enterprise
and Splunk Universal Forwarder to monitor Windows Security Events.

## Current Capabilities
- Windows Event Log collection (Security, System, Application)
- Splunk dashboards for authentication monitoring
- Detection of security-related events
- Hands-on SOC fundamentals

## Tools Used
- Splunk Enterprise
- Splunk Universal Forwarder
- Windows Event Logs
