## Current Detections

Implemented detections are based on Windows Security Event IDs:

- Successful Logon (4624)
- Failed Logon (4625)
- Account Lockout (4740)
- New User Created (4720)
- User Added to Admin Group (4732)
- Privilege Change Attempt (4670)

Detections rely on real system activity.
No simulated data is used.
