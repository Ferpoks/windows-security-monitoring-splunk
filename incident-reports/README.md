# Incident Reports

This folder contains hands-on incident response reports based on Windows Security Event Logs collected in the Splunk mini-SIEM lab.

## Reports
- **IR-001** – Brute Force Attempt Followed by Successful Login  
  Focus: Event IDs 4625 → 4624 | MITRE: T1110

- **IR-002** – Suspicious PowerShell Execution  
  Focus: PowerShell abuse / encoded commands | MITRE: T1059.001

- **IR-003** – New Admin User Created  
  Focus: Account creation & privilege escalation | MITRE: T1136
