## Windows Security Monitoring Dashboard

This dashboard includes:

- Successful Logons (EventCode 4624)
- Failed Logons (EventCode 4625)
- Top Logged Users
- Privilege Escalation Indicators
- New Local Admin Created
- Remote Desktop Logons

The dashboard was built using:
- Splunk Dashboard Studio
- Grid Layout
- Single Value and Chart Panels

Some panels show no data because the security events have not occurred yet.
